## Gomoku Monte Carlo Automatic Strategy Model

**wuziqi** is a Monte Carlo automatic strategy model based on a special competition board (circular) and rules. Our model won the competition in the Gomoku confrontation at HKUST. You are welcome to experience it using `finalplayer.py`, which includes four different machine models with varying "intelligence levels" (`computer_move1` to `computer_move4`), with increasing difficulty. You can play against them or import your own Gomoku strategy for automatic matches ğŸ˜Š.

For detailed rules, please refer to `Final Projectchess.html` or `Final Projectchess.ipynb`.
